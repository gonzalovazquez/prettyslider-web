Pretty Slider

A beautiful jQuery-less slider implementation for AngularJS.

{{ item.cost }}

But how do I use it?

Oh, that's the easy part.

  1. Make sure you have AngularJS v1.1.4 or higher installed. If you don't yet, this is a good tutorial to get started.
  2. Clone the repo!
  3. Make sure to load AngularJS first in your HTML, followed by angular-slider.js file. Don't forget to include the related angular-slider.css file.
  4. The module name is named prettySlider. To enable it, you must simply list it as a dependency in your app. Example:
var app = angular.module('app', ['prettySlider', 'ngResource', ...]);
  1. You can then use it in your template like so:
 <div pretty-slider floor="0" ceiling="1000" step="50" precision="2" ng-model="item.cost">
